Adminer
adminer.selfmade.solution

Adminer (formerly phpMyAdmin) is a full-featured database management tool written in PHP.

  • Adminer is a lightweight, open-source web-based database management tool written in PHP.
  • It's a single-file application, making it easy to deploy and use as a phpMyAdmin alternative.
  • Supports multiple database systems, allowing users to manage different databases from a single interface.
