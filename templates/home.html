Adminer
adminer.selfmade.solution

Adminer (formerly phpMyAdmin) is a full-featured database management tool written in PHP.

  • Adminer is a lightweight, open-source web-based database management tool written in PHP.
  • It's a single-file application, making it easy to deploy and use as a phpMyAdmin alternative.
  • Supports multiple database systems, allowing users to manage different databases from a single interface.
    • MYSQL SERVER
    adminer.selfmade.solution

    MySQL is the world's most popular open source database.

  • MySQL is an open-source relational database management system (RDBMS).
  • Based on Structured Query Language (SQL).
  • Widely used in web applications (e.g., WordPress, PHP apps) and enterprise systems
  • Optimized for read-heavy workloads.
    • MONGODB SERVER
    adminer.selfmade.solution

    MongoDB is a source-available cross-platform document-oriented database program.

  • MongoDB is a NoSQL, open-source document-oriented database.
  • Stores data in JSON-like BSON (Binary JSON) format.
  • Designed for high scalability, flexibility, and performance.
  • Suitable for real-time big data apps.
    • MARIADB SERVER
    adminer.selfmade.solution

    MariaDB Server is one of the most popular MySQL fork in the world.

  • MariaDB is an open-source relational database management system (RDBMS).
  • It is a fork of MySQL, created by the original MySQL developers.
  • Often faster than MySQL due to better optimization.
    • PostgreSQL SERVER
    adminer.selfmade.solution

    PostgreSQL is a powerful, open source object-relational database system with over 30 years of active development

  • PostgreSQL (often called Postgres) is a powerful, open-source object-relational database system.
  • Allows users to define custom functions, data types, operators, and even languages.
  • Fully supports Atomicity, Consistency, Isolation, and Durability.
